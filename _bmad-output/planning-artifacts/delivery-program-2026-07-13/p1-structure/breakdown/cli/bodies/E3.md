## 🎯 Epic — Wiki Namespace on orvex-wiki-api

The `wiki` namespace is the largest verb surface and the product's history — but here it is **one namespace among peers**, routed **domain-pure** to `orvex-wiki-api` for *every* wiki-shaped access, including surfaces that were engine-direct in the reference (history/diff/version, bulk import/export, binary attachment I/O, drift, audit dual-write, comment CRUD). The engine's raw API is never contacted (D-S16). This epic delivers the full grammar — page CRUD + the verb family (`get/list/create/update/patch/upsert/replace/edit/scaffold/delete/trash/purge/move/restore/duplicate`) with CAS `ifVersion→409 VERSION_MISMATCH`, `DUPLICATE_CANDIDATE` (exit 8), and `QUOTA_EXCEEDED` propagated verbatim; nav/watchers/permissions/transclusion/governance; block authoring for the **21 retained schema block types + the `diagram` command** (Linear's 8 dropped); `space`/`comment`/`label`/`attach`; `migrate`; the `verify` suite + `spec gate check`; and the full-binary `screenshot`/`code graph` verbs.
