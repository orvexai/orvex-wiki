## 🎯 Feature area

This is the payoff epic: deliver a Memory-enriched composed prompt **into the ChatGPT and Gemini web UIs without copy/paste** (FR-D1), by injecting into the user's *own* compose surface (FR-D5), covering the beachhead set ChatGPT/Claude/Gemini/Grok with honest not-yet-supported messaging for the rest (FR-D3).
