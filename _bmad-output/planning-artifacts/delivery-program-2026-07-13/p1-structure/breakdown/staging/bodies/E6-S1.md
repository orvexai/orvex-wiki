## 🎯 Story

As the **Librarian learning subsystem** I want **every human accept/reject/edit on a recommendation recorded as an append-only Feedback Event with optional reason, linked to the Proposal, Disposition, and the exact prompt-pack version that produced it** so that **the learning signal is complete, attributable, and never re-reads the currently-active pack**.

**Definition of Done:** one named test `TestFeedbackEventLinksProducingPackVersion` (integration — a decision writes an append-only Feedback Event carrying the decision, optional reason, Proposal id, Disposition id, and the pack version that produced that Disposition — not the current active pack; the event is immutable). *Final H1–H17 elaboration + exact contract tag/versions are pinned at pack certification (ENG-2094); this story is dispatch-blocked until that tag exists.*
