## 🎯 Story

As a **user**, I want a **Studio-styled native read pane, hover cards, and activity feed — all acting as me**, so that I can read my wiki pages inside Studio under my own permissions, with every wiki write and user-scoped read carrying a short-lived delegated user token.

**Definition of Done:** one named test **`TestActAsUserTokenNeverPersisted`** (integration layer — renders a wiki page in the read pane and performs a quick-capture write, asserting both carry an identity-minted delegated user token that is short-lived, audience-scoped to the target service, and never persisted/cached/logged, and that studio-api holds no elevated wiki credential on user-facing paths, through the read-pane + capture interfaces). *Final H1–H17 elaboration + exact contract tag/versions are pinned at pack certification (ENG-2098); this story is dispatch-blocked until that tag exists.*

## ✅ Acceptance Criteria
- [ ] **AC1** — Given a wiki page, When rendered in the read pane, Then it is Studio-styled read-only, fetched through the DocmostPort under the user's permissions (own space + demo wikis only). *Assertion: read pane renders own-space pages; other-space → denied.* [Source: FR-SA25]
- [ ] **AC2** — Given an unsupported embed, When rendered, Then it degrades to a labeled open-in-wiki block. *Assertion: unsupported embed → labeled fallback, not a broken render.* [Source: FR-SA25]
- [ ] **AC3** — Given the activity feed + quick capture (FR-60), When used, Then Free capture writes direct via the gate's non-AI path and £7 routes per write-mode. *Assertion: Free quick-capture → non-AI gate; £7 → write-mode routing.* [Source: FR-SA25]
- [ ] **AC4** — Given any wiki write or user-scoped read, When performed, Then it carries an identity-minted delegated user token; studio-api holds no elevated wiki credential on user-facing paths. *Assertion: user-facing wiki call uses a delegated token, never a service credential.* [Source: FR-SA26]
- [ ] **AC5** — Given delegated tokens, When used, Then they are short-lived, audience-scoped to the target service, and never persisted/cached/logged (and never ride workflow inputs). *Assertion: token grep finds no persistence/cache/log; TTL bounded.* [Source: FR-SA26, NFR-SA2]
- [ ] **AC6** — Given "Edit in wiki ↗", When invoked, Then it hands off via the engine-minted magic-link OTP (transported, not minted here). *Assertion: hand-off uses engine OTP; studio-api mints no OTP.* [Source: FR-SA26]
