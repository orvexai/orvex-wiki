- [ ] **AC1** — Given a built `cmd/console` binary with `CELL_ID` and `CLUSTER_NAME` injected, When GET `/healthz`, Then 200 with both values as **distinct** params. *Assert: JSON body has `cell_id`==env CELL_ID and `cluster_name`==env CLUSTER_NAME, non-equal keys.* [Source: Arch §3, cell-contract rule 4]
- [ ] **AC2** — Given no auth header, When GET `/healthz`, Then it still serves (the only unauthenticated route). *Assert: 200 with no `Authorization` header.* [Source: Arch §3]
- [ ] **AC3** — Given `CELL_ID` unset in a prod profile, When the binary boots, Then it fails fast (no baked default). *Assert: non-zero exit + loud config error; no fallback value.* [Source: Map — no-fallbacks hard cut; cell rule 4]
- [ ] **AC4** — Given the binary serves both the SPA mount and `/api`, When it starts, Then one process listens on one port. *Assert: single HTTP server, `/` and `/api` from one binary.* [Source: Arch §1 workload]
