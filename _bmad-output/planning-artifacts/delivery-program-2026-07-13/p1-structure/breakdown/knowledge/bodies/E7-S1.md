## 🎯 Story
As the Orvex CLI cache-sync I want a cursor/delta projection-download API ("everything in space X changed since cursor C") that is ACL-narrowed, epoch-bearing, and sustains ≥ 20 pages/s so that cache fill replaces the O(pages) engine crawl at ≥ 5× speed.
