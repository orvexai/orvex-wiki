## 🗺️ Feature area

The app shell is the frame every surface mounts inside, and the first-run state machine is what makes Orvex feel alive in the first 90 seconds (the brief's Priya bar). Grounded in the First-Run + App-Shell spec (`KeREQUiGjA`), this epic delivers the signed-in shell (6-destination sidebar, the single ⊕ Create CTA, account footer, ⌘K palette, inbox drawer, responsive desktop/tablet/mobile), the ANON/NEW-AUTH/RETURNING state machine that resolves from the Clerk session + BFF `memoryState` and mounts the right chrome, the public front door + route map with client-side signed-in redirect and deep-link restore, and the RETURNING/NEW-AUTH dashboard with honest-empty zones that fill live on state transitions. It exists because the PRD's honesty rules (G1, R3/R4/R5) and the no-org-chrome single-tenant Alpha posture (D-S17) are shell-level invariants — get the frame wrong and every child surface inherits the drift.
