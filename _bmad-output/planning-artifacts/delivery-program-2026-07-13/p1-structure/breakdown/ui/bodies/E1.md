## 🗺️ Feature area

The shared substrate every other UI epic stands on. The evidence digest is blunt: the repo is a single-commit scaffold with **no test runner, no eslint, no data-fetch lib, no contracts-codegen dep, no CI**, and its serving posture (orange-proxied nginx origin, prod sourcemaps) contradicts canon. This epic lands the Wave-0 foundations so the ~35 product surfaces can be built in isolation against pinned fixtures: the contracts-codegen client + identity session-token attach (A-STATELESS/A-SHELL), the exact-match `orvex-ds.css` token pipeline with theme-before-first-paint (A-DS), the CI ship-gate harness (vitest+Playwright+axe+visual+token-lint) enrolled in the `cell-lint` required status, the shared **draft-vs-committed** proposal layer (FR-UI27/A-DRAFT) implemented once so it cannot fork per surface, the honest-states primitives (FR-UI29), the single SSE entitlement channel + over-quota 402 rendering (FR-UI28/NFR-UI10/A-AUTH), and the Workers-Static-Assets region-agnostic serving topology (A-GLOBAL/NFR-UI1). These exist because the PRD's hard ship-gates (G5, NFR-UI2/UI3) and the one-upstream/no-secrets invariants are unenforceable until the tooling and shared layers exist.
