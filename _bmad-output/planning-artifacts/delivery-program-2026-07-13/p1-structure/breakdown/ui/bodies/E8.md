## 🗺️ Feature area

The cross-cutting NFR ship-gates and instrumentation that make the whole SPA honest, fast, secure, and observable — the gates the SE-Arch audit flagged as **unlanded** despite being "hard gates" in the PRD. This epic delivers the WCAG 2.1 AA ship-gate across both themes AND both skins (axe + token-pair contrast + reduced-motion snapshots, ≥48×48 targets, focus-ring/traps, aria-live regions), the performance/scale/streaming-resilience/offline posture (first-load JS budget, code-split, no-CLS reserved space, 1k/10k virtualization ≤200ms p95, SSE disconnect tolerance, offline weave/cached Copy), the security + BFF-only egress hardening + observability (no secrets in bundle, CSP allowlist, XSS-sanitize remote content, CI egress guard, web-vitals + JS error reporting + build metadata), and the product-metric SM funnel instrumentation to the BFF ingest (first-party only). It exists because these invariants cannot be bolted on at the end (the audit's warning) — they are CI-enforced gates that must be green before any surface counts as delivered ("looks good AND works").
