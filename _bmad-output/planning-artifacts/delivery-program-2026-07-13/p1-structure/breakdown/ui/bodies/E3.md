## 🗺️ Feature area

Memory is the flagship "your AI finally knows you" surface — the brief calls it the portrait of what Orvex knows about you that "must look good *and* work." This epic delivers the streaming FormSpec onboarding (the wow: AI builds the form, the user fills the values — never AI-invented facts, R2) with its byte-identical deterministic fallback (FR-1b), the user-initiated form→Home morph choreography, the private Memory wall with full FR-31 provenance, the sensitive-field consent UX with session-only buffering and prominent hard-delete, and the Memory Coach entitlement gating plus 1k/10k scale + Professional Mode skin. It exists because Memory carries three hard invariants at once: the draft-vs-committed rendering guarantee (G3), privacy first-classness (R1, SM-C3 = 0 leaks), and the streaming budgets (SM-9: TTFT ≤2.5s, first blocks <3s, full <12s) that make or break the first impression.
