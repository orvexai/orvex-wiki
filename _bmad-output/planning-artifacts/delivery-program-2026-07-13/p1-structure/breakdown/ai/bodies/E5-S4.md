As the last hop before LiteLLM, I want a single outbound middleware running the per-tenant PII redactor + secret-scrubber on every model-bound payload (prompts, context, tool arguments, retrieved ask-loop chunks), failing closed on the strictest policy when policy lookup fails, so that no feature path can forget redaction (FR-AI20, A-PII).
