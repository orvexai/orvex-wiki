These are the seams every isolated build compiles against"* (current-state-map P0 #2). Today the repo has zero wiki-api/ai/billing entries filed — most *"contracts"* are prose in each service's arch page, not enforceable artifacts (risk #1). This epic turns that prose into pinned OpenAPI.
