## 🎯 Story
As a consumer that codegens typed clients, I want the versioning & change policy realized — `VERSION`, `CHANGELOG.md`, repo-level semver tags as the consumption unit, per-artifact `x-status: draft|pinned` markers, and the first immutable tag `v0.1.0` — so that I pin one coherent contract set that is reproducible from a tag alone with no network or engine checkout.

**Definition of Done:** ONE named test `TestCodegenReproducibleFromTag` — a CI-layer test asserting, through the release tooling, that a checkout of tag `v0.1.0` (no network, no engine repo) produces byte-identical codegen output on two runs, and that `cell-lint.yml` references an immutable tag rather than `@dev`. *Final H1–H17 elaboration + exact contract tag/versions are pinned at pack certification (ENG-2091); this story is dispatch-blocked until that tag exists.*

## ✅ Acceptance Criteria
- [ ] **AC1** — Given a release, When `VERSION` + `CHANGELOG.md` land, Then CI release-gates them (no tag without a CHANGELOG entry). *Assert: release job fails on missing CHANGELOG delta* [Source: FR-C17, A-VERSION].
- [ ] **AC2** — Given an additive change, When classified, Then it is a minor bump; a breaking change is major with a documented dual-publish/deprecation window + breaking-change notice. *Assert: semver classifier maps additive→minor, field-removal→major* [Source: FR-C17, ADR-0008].
- [ ] **AC3** — Given tag `v0.1.0`, When a consumer codegens twice offline, Then output is byte-identical. *Assert: `sha256(run1) == sha256(run2)`, no network syscalls* [Source: NFR-C2].
- [ ] **AC4** — Given `cell-lint.yml` floating on `@dev` (negative path), When the first tag exists, Then consuming CI is flipped to `@<immutable-tag>` and `@dev` is forbidden post-v0.1. *Assert: no `@dev` ref remains in consuming CI* [Source: SE-Arch T5, A-GATES].
- [ ] **AC5** — Given a `pinned` artifact, When someone edits it non-additively without the process, Then the change policy blocks it. *Assert: pinned-artifact breaking edit fails the classifier* [Source: A-VERSION].
