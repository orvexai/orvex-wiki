## 🎯 Story

**Audit: already implemented** (evidence in §4) — this story is VERIFY + harden: prove the behaviour via the named DoD test + adversarial review; do not rebuild.

**As the** platform, **I want** every upstream call — wiki-api, ai, knowledge, and the Studio backend alike — to carry the caller's identity-minted scoped token **verbatim**, with the MCP holding no service credential on any user-facing path and never exchanging a caller token for anything, **so that** a scoped caller can never be amplified and a token can do through the MCP only what it could do directly.

**Definition of Done:** one named test `TestTokenPassThroughNoElevatedCredentialOnAnyUpstream` (integration — assert each upstream call carries the caller's Authorization header unchanged, that no service credential/placeholder Studio JWT is ever attached, and that absent/invalid/insufficient tokens fail closed). *Final H1–H17 elaboration + exact contract tag/versions are pinned at pack certification (ENG-2102); this story is dispatch-blocked until that tag exists.*
