## Feature area

The G2 lossless-lifecycle guarantee. The as-built 10-attempt AUTHORITATIVE cap (~5-minute budget) permanently failed a lifecycle workflow after any longer callee outage *past the webhook ACK* — worst case the irreversible 720-hour purge fired into a brief outage and never ran. This epic corrects AUTHORITATIVE to unbounded (`MaximumAttempts:0`), fixes the already-started NACK loop, adds `spec.delivery` retry/backoff + `deadLetterSink` DLQ to every Trigger, routes malformed/unsupported events to the DLQ (DLQ-as-record), and wires long-retry + workflow-failure alerting with an operator-replay runbook.

**Why it exists:** PRD G2/G3 + NFR-W5/NFR-W7 + FR-W6 + FR-W15; Arch A-W1/A-W4/A-W8 + §5 risks. 'Ship it before anything else in the hardening wave' (§5). Temporal history + DLQ = this service's record (satisfies identity A-DATA without a datastore).
