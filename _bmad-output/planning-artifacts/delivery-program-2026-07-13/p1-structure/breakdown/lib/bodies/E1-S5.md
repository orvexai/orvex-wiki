## 🎯 Story

**Audit: already implemented** (evidence in §4) — this story is VERIFY + harden: prove the behaviour via the named DoD test + adversarial review; do not rebuild.

**As** the family, **I want** one `net/http` router that enforces deny-by-default scope on every protected route, a `WorkloadIdentityOnly` route class for scope-less internal surfaces, and a distinct cell-guard 421 middleware, **so that** a scoped token can never read outside its scope in any satellite, no satellite can opt out, and a foreign-cell request is refused with 421 before any handler runs (FR-L6; Arch A-VERIFY-IMPL step 6, A-CELL Rule 3, §3 V4/V11/V12).

**Definition of Done:** ONE named test `TestRouterDenyByDefaultAndCellGuard` (unit, in `pkg/auth`, driving requests through the mounted router) asserting: an undeclared route rejects everything; an over-scope request is rejected before the handler; a `WorkloadIdentityOnly` route rejects bearer tokens and a scoped route rejects workload-only callers; and `principal.cell != CELL_ID` ⇒ 421 (distinct from a 403 scope reject). *Final H1–H17 elaboration + exact contract tag/versions are pinned at pack certification (ENG-2092); this story is dispatch-blocked until that tag exists.*

## ✅ Acceptance Criteria

- [ ] **AC1** — Given a route with no scope declaration, When any request arrives, Then it is rejected (deny-by-default) at startup/CI where detectable and at request time regardless. *assert undeclared route ⇒ reject-all + router-coverage lint failure* [Source: FR-L6 / Arch V4]
- [ ] **AC2** — Given a token whose effective scope does not satisfy the route's declared required scope, When a request arrives, Then it is rejected **before handler code runs**. *assert handler never invoked; 403* [Source: FR-L6 / Arch V4]
- [ ] **AC3** — Given a `WorkloadIdentityOnly` route, When a bearer token is presented, Then it is rejected; Given a scoped route, When only workload identity is presented, Then it is rejected. *assert both cross-cases reject (V11)* [Source: FR-L6 / Arch V11]
- [ ] **AC4** (cell) — Given `principal.cell != CELL_ID`, When a request arrives, Then the cell-guard returns **421, never served** — a distinct step from the scope gate. *assert 421 (not 403); every mounted route passes the guard or router-coverage lint fails* [Source: FR-L6 / cell Rule 3, Arch V12]
- [ ] **AC5** (solo) — Given the `solo` sentinel cell with no registry endpoint configured, When a request arrives, Then the cell-guard no-ops so a self-hosted engine boots. *assert guard passes on solo* [Source: cell Rule 3 / Arch A-CELL]
