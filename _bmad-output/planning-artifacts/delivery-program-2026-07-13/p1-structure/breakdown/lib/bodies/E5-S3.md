## 🎯 Story

**As** knowledge's SSE gateway (the first consumer) and any offset-based streaming reader, **I want** `pkg/kafkautil` — standalone **assigned** consumers (`assign()` + `seek`, no shared group so every pod sees all events), read-from-offset / tail-follow / bounded replay, and `<topic-epoch>:<offset>` cursor helpers where an unparseable cursor or retention-expired offset resolves to a typed `ErrGap` — over `{domain}-events.{cell}` single-partition, cell-opaque topics, **so that** the Kafka-first spine is consumable day one without silent skipping and without any code ever parsing a cell id (FR-L22; Arch A-KAFKA, cell Rule 5).

**Definition of Done:** ONE named test `TestAssignedConsumerCursorGap` (unit, in `pkg/kafkautil`, driving an assigned consumer against a fake/in-memory Kafka) asserting `assign()`+`seek` (never a shared group), that a retention-expired or unparseable cursor resolves to `ErrGap` (never an error page, never a silent skip), and that topic names are cell-opaque (never substringed/parsed). *Final H1–H17 elaboration + exact contract tag/versions are pinned at pack certification (ENG-2092); this story is dispatch-blocked until that tag exists.*
