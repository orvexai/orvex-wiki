## 🎯 Story

**Audit: already implemented** (evidence in §4) — this story is VERIFY + harden: prove the behaviour via the named DoD test + adversarial review; do not rebuild.

As a **quota enforcer** I want **an entitlement-reader interface that reads per-tenant billing-owned entitlements (cached, short-TTL, evicted on `billing.entitlement.changed`), with an interim hardcode-Free constant behind the same interface** so that **the engine hard-codes no plan numbers yet quota enforcement isn't blocked on billing being built** (FR-W11).

**Definition of Done:** ONE named test `TestEntitlementReaderEvictsOnBillingChange` (integration layer — asserts through the entitlement-reader interface that a cached entitlement is served within TTL, evicted on a `billing.entitlement.changed` event, and that the interim Free constant is served only via the interface). *Final H1–H17 elaboration + exact contract tag/versions are pinned at pack certification (ENG-2103); this story is dispatch-blocked until that tag exists.*

## ✅ Acceptance Criteria

- [ ] **AC1** — Given a tenant, When the reader is called, Then it returns the effective entitlement (`max_pages`, `storage_bytes_aggregate`, `max_file_bytes`, `max_files`, `max_members`, `history_retention`, AI spend cap) from a short-TTL cache. *Assert: fields present; second call within TTL hits cache.* [Source: FR-W11]
- [ ] **AC2** — Given a `billing.entitlement.changed` event, When received, Then the cached entitlement is evicted. *Assert: next read re-fetches.* [Source: FR-W11, OQ-W2]
- [ ] **AC3** — Given the free-only launch, When billing SoR is absent, Then the interim Free constant (200 pages / 1 GiB / 10 MB-file / 2,000 files / 25 members / `min(10,180d)`) is served behind the interface. *Assert: no plan numbers outside the reader; constant swappable.* [Source: FR-W11]
- [ ] **AC4 (error-path)** — Given the interim constant, When any paid plan becomes sellable, Then the reader MUST read the billing SoR (swap point). *Assert: guard/flag prevents selling a paid plan while hardcoded; TBD — pinned by the Definition Pack (ENG-2103) for the exact contract shape OQ-W2.* [Source: FR-W11]
- [ ] **AC5 (negative)** — Given engine code, When grepped, Then no plan number appears outside the entitlement-reader. *Assert: grep finds none.* [Source: FR-W11]
