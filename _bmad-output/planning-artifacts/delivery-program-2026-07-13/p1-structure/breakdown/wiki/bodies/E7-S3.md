## 🎯 Story

As a **billing/identity peer** I want **the polymorphic user-or-org tenant model (a personal user gets NO Clerk org — a user-keyed tenant with a full workspace + entitlements; Teams get a real Clerk org), the personal→Teams upgrade-pass that re-keys the tenant in place, and global-uniqueness delegation to the identity registry** so that **Clerk per-org MAU cost is avoided at 10k–100k scale, upgrades carry data + entitlements without a migration, and two cells never both mint the same tenant** (A-TENANCY, D-S17).

**Definition of Done:** ONE named test `TestPersonalToTeamsUpgradePassReKeysInPlace` (integration layer — asserts through the upgrade-pass that a user-keyed tenant becomes org-keyed while `workspace_id` stays the RLS tenant key and data + entitlements carry over, seats begin counting). *Final H1–H17 elaboration + exact contract tag/versions are pinned at pack certification (ENG-2103); this story is dispatch-blocked until that tag exists.*

## ✅ Acceptance Criteria

- [ ] **AC1** — Given a personal user, When a tenant is provisioned, Then it is a user-keyed tenant with a full wiki workspace + entitlements and NO Clerk org. *Assert: `{user}` principal; no org minted.* [Source: A-TENANCY]
- [ ] **AC2** — Given a Team, When provisioned, Then it gets a real Clerk org keyed on the `{org}` principal; billing/identity key on the same polymorphic principal. *Assert: `{org}` principal consistent across billing/identity.* [Source: A-TENANCY]
- [ ] **AC3** — Given a personal→Teams upgrade, When the upgrade-pass runs, Then it mints the org, re-keys the tenant in place, carries data + entitlements, and seats start counting — with `workspace_id` unchanged as the RLS tenant key. *Assert: `workspace_id` stable; data + entitlements preserved.* [Source: A-TENANCY]
- [ ] **AC4** — Given tenant/hostname/slug reservation, When minting, Then it delegates to the GLOBAL registry (identity is sole writer); the engine's local `findByHostname` unique index is only a per-cell backstop. *Assert: two cells cannot both mint `acme`; engine index is backstop-only.* [Source: A-TENANCY]
- [ ] **AC5 (error-path)** — Given a mint collision across cells, When attempted, Then the global registry rejects it. *Assert: cross-cell duplicate rejected at the registry, not silently accepted locally.* [Source: A-TENANCY]
