- [ ] **AC1** — Given a page at version N, When `apply-ops` is called with `ifVersion=N` and a valid typed PM-JSON op batch, Then it applies atomically and bumps the version. *Assert: version==N+1; ops applied; restorable history entry written.* [Source: FR-W1]
- [ ] **AC2** — Given a page at version N, When `apply-ops` is called with `ifVersion=N-1` (stale), Then it returns 409. *Assert: HTTP 409, no mutation.* [Source: FR-W1]
