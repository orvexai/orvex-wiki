### 🎯 Story
As a **planner agent handing work to a builder on another platform** I want **a first-class handoff with a typed payload whose state derives from `(status, assignee, resolution)` and whose target resolves against identity-registered roles** so that **context transfers cleanly with zero loss and routing never fragments on free-form strings**.
