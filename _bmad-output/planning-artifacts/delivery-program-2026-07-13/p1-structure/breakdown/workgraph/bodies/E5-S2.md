### 🎯 Story
As a **platform owner** I want **tiered compaction (tier-1 LLM summarize / tier-2 deep archive) that is reference-aware and restorable** so that **closed items past their window shrink the store without gutting context an active item depends on**.

- [ ] **AC2** — Given tier-2, When an item passes the deep window (default 90d), Then it deep-archives. *assert: tier-2 archive applied.* [Source: FR-MEM12]
- [ ] **AC3** — Given an item cited by a non-closed item, When compaction considers it, Then it is protected OR its summary preserves resolution of the cited short id. *assert: cited id resolvable post-compaction; uncited item compactable.* [Source: FR-MEM12, AD-5]
- [ ] **AC4** — Given a compacted item, When restore is requested, Then the original is recoverable from history. *assert: restore returns the pre-compaction content.* [Source: FR-MEM12, NFR-MEM5]
