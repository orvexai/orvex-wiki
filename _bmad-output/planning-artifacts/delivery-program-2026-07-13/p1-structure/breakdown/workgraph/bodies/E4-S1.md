### 🎯 Story
As a **coordinating agent** I want **tenant- and agent-scoped Remembered Facts via `remember`/`recall`/`forget` with injection caps and explicit elision banners** so that **persistent facts survive sessions and truncation is always announced, never silent**.

**Definition of Done:** one named test `TestRecallCapsWithElisionBanner` (domain layer) asserts recall enforces max-count/max-chars caps, always returns at least one fact or an explicit empty-with-status, and attaches the `{kind: elision, scope, detail}` banner whenever anything is elided. *Final H1–H17 elaboration + exact contract tag/versions are pinned at pack certification (ENG-2095); this story is dispatch-blocked until that tag exists.*

### ✅ Acceptance Criteria
- [ ] **AC1** — Given `remember`, When called, Then a fact persists keyed unique on `(tenant, scope ∈ {tenant|agent|session}, key)`; admin can pin tenant facts. *assert: unique key enforced; pinned flag settable by admin.* [Source: FR-MEM8, AD-9]
- [ ] **AC2** — Given `recall`, When resolving a key, Then precedence is `session → agent → tenant`; injection caps (max-count, max-chars) apply. *assert: resolution order correct; caps enforced.* [Source: FR-MEM8, AD-9]
- [ ] **AC3** — Given caps truncate results, When recall returns, Then it carries an explicit elision banner (never silently dropped). *assert: banner present with scope+detail.* [Source: FR-MEM8, AD-13]
- [ ] **AC4** — Given no matching fact, When recall runs, Then it returns an explicit empty-with-status (always at least one memory or explicit empty). *assert: empty-with-status, not a bare null.* [Source: FR-MEM8]
- [ ] **AC5 (negative)** — Given a memory-key write that looks like a read, When submitted, Then it is treated as a read (beads guard). *assert: read-shaped remember is not a destructive write.* [Source: Architecture Agent-safety]
- [ ] **AC6 (error-path)** — Given `forget`, When it targets a fact, Then it soft-deletes with a trail (hard erasure is FR-MEM13 / E5-S3). *assert: forget leaves a decision trail.* [Source: FR-MEM8, NFR-MEM5]
